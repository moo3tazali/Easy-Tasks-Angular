# Easy-Tasks-Angular

This project is a basic to-do application built with Angular V18.

## Features

1. **Task Management**

   - Add new tasks.
   - Edit existing tasks.
   - Delete tasks.
   - Mark tasks as completed.

2. **User Interface**

   - Responsive design.
   - Intuitive task list display.
   - Visual indicators for task status (completed/pending).

3. **Data Persistence**

   - Local storage for saving tasks.

4. **Components**
   - Task list component.
   - Task item component.
   - Task form component.

## Getting Started

### Prerequisites

- Node.js
- Angular CLI

### Installation

1. Clone the repository:

   ```sh
   git clone https://github.com/moo3tazali/Easy-Tasks-Angular.git
   cd Easy-Tasks-Angular

   ```

2. Install dependencies:
   ```sh
   npm install
   ```

### Running the App

1. Start the development server:

   ```sh
   ng serve
   ```

2. Open your browser and navigate to `http://localhost:4200/`.

### Building the App

To build the project, run:

```sh
ng build
```

The build artifacts will be stored in the `dist/` directory.

### Running Tests

- Unit tests:

  ```sh
  ng test
  ```

- End-to-end tests:

  ```sh
  ng e2e
  ```

### Further Help

For more help on Angular CLI, use:

```sh
ng help
```

Visit the [Angular CLI Overview and Command Reference](https://angular.io/cli) page for detailed documentation.
